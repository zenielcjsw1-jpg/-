=========================================
인원현황판 Lite v1.0
=========================================

■ 구성 파일

index.html
style.css
script.js
images/warehouse.png


=========================================
실행 방법
=========================================

1. 모든 파일을 같은 폴더에 둡니다.

2. images 폴더 안에
warehouse.png 파일을 넣습니다.

3. index.html을 실행합니다.

별도의 설치는 필요 없습니다.


=========================================
사용 방법
=========================================

1. 직원 이름을 손가락(또는 마우스)으로
원하는 위치로 이동합니다.

2. 저장 버튼을 누르면
현재 위치가 저장됩니다.

3. 다음 실행 시
마지막 저장 위치가 자동으로 불러와집니다.

4. 초기화 버튼을 누르면
기본 위치로 돌아갑니다.
